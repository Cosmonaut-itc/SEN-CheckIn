Paleta Michoacán

Un rediseño radical. Adiós a los tonos tierra conservadores. La nueva paleta de SEN nace de Michoacán: el cobre martillado de Santa Clara, el verde profundo de los aguacatales, la guinda de las lacas de Pátzcuaro, y el verde sereno de los bosques de oyamel.

Filosofía: Software empresarial no tiene por qué ser gris y aburrido. Si México tiene los colores más vibrantes del mundo, ¿por qué nuestras herramientas deberían verse como si fueran de Estocolmo? SEN se ve como se siente: mexicano, orgulloso, moderno.

Inspiración de Michoacán

Cada color tiene raíces reales en la cultura, geografía y tradiciones de Michoacán y Morelia.

🌺 Cobre Michoacano
Del cobre martillado artesanal de Santa Clara del Cobre, pueblo mágico de Michoacán. No es un naranja tímido — es el bronce cálido que evoca la tradición purépecha.
🌌 Verde Aguacate
De los aguacatales infinitos que cubren los cerros michoacanos. Un verde profundo y vivo que evoca abundancia, tierra fértil y el motor económico del estado.
🌵 Guinda Pátzcuaro
De las lacas de Pátzcuaro y las artesanías purépechas de Uruapan. Una guinda profunda que conecta con la tradición textil y decorativa de los artesanos michoacanos.
🌿 Verde Bosque
De los bosques de oyamel de las montañas michoacanas donde llegan las mariposas monarca. Un verde sereno y profundo para estados de éxito y confirmaciones.

Comparación: antes → después

El cambio es radical. De una paleta tierra conservadora a una identidad vibrante y orgullosamente mexicana.

✗ v3 — Antes
Tonos tierra apagados. Terracota + sage. Neutros grises. Conservador, seguro, anónimo. Podría ser cualquier app "orgánica".
✓ v4 — Ahora
Cobre michoacano cálido. Verde aguacate profundo. Guinda pátzcuaro. Neutros con toque cantera. Inconfundiblemente mexicano.

Reglas de uso

✓ Así sí
Cobre Michoacano como color primario de acción (CTAs, links, toggles activos). Verde Aguacate para texto y elementos informativos. Guinda Pátzcuaro para highlights y badges especiales. El texto primario es caoba oscuro (#2B1810), nunca negro puro.
✗ Así no
Nunca uses rosa y error rojo uno al lado del otro — son similares y confunden. Nunca uses los tres acentos juntos sin jerarquía. El guinda pátzcuaro NO es para texto — es solo para pequeños acentos y badges.
✓ Dark mode
Los fondos oscuros tienen undertone tierra (#110D0A), no gris neutro. Los acentos se iluminan (cobre→#D4835E, verde→#7FB573, guinda→#C85A8A) para mantener WCAG AA.
✗ Dark mode
Nunca uses fondos grises puros (#141414, #1C1C1C). Pierden toda la personalidad de la paleta. El undertone cálido es lo que hace que el dark mode se sienta SEN y no genérico.

Filosofía

Una identidad, respetuosa de cada plataforma móvil

SEN mantiene su personalidad (colores vibrantes de Michoacán, tipografía, voz) en iOS y Android, pero respeta las convenciones nativas de cada sistema: HIG para iOS, Material Design 3 para Android. Los usuarios esperan comportamiento consistente con su OS — nosotros nos adaptamos.

Regla de oro: Usa los tokens SEN para color, tipo y espaciado. Usa el comportamiento nativo de cada plataforma para navegación, gestos e interacciones.
SEN Lo que compartimos

Colores (cobre michoacano, verde aguacate, guinda pátzcuaro, paleta completa). Tipografía (Playfair + DM Sans). Voz y tono (tuteo, directo, mexicano). Espaciado (escala de 4). Iconografía semántica.

iOS Lo que adaptamos

Tab bar abajo con labels. Navigation con large titles. Sheets con detents. Swipe back nativo. Haptics con UIFeedback. Dynamic Type.

ANDROID Lo que adaptamos

Navigation bar M3 abajo. Top app bar con scroll behavior. Bottom sheets M3. Back predictivo. Haptics con VibrationEffect. Material motion.

✓ Regla
iOS body = 17pt (estándar HIG). Android body = 16sp (estándar M3). No uses 16 en iOS ni 17 en Android — respeta cada sistema.
✗ Evita
Un solo valor para ambas plataformas. 16 en iOS causa texto que se ve apretado vs apps nativas. 17 en Android se siente sobredimensionado.
Dynamic Type / Font Scaling: Nunca uses allowFontScaling: false en React Native. iOS escala desde 17pt hasta ~53pt en los tamaños más grandes. Android escala hasta 200%. Tu layout debe aguantarlo.

Uniwind + HeroUI Native

SEN Mobile usa Uniwind (Tailwind CSS v4 para React Native) como motor de estilos y HeroUI Native como librería de componentes. Ambas se integran nativamente y comparten el mismo sistema de tokens via global.css.

Offline First

Las oficinas y sucursales no siempre tienen buena señal. El producto debe funcionar offline para las operaciones básicas.

✓ Funciona offline
Ver personal actual, crear nómina (queue local), escanear empleados, buscar en catálogo local. Indicador sutil "sin conexión" en top.
✗ Requiere conexión
Sincronización con otras sucursales, generar reportes, exportar PDFs. Mostrar mensaje claro: "Necesitas conexión para sincronizar."

Checklist pre-release

Antes de publicar cualquier pantalla en Jale, verifica estos puntos.

☐ Tokens: Todos los colores usan clases Uniwind (bg-primary, text-foreground), no hex directo
☐ Dark mode: Probado en ambos temas. Toggle funciona
☐ iOS body = 17pt, Android body = 16sp
☐ Touch targets: iOS ≥ 44pt, Android ≥ 48dp
☐ Safe areas: Contenido no tapa notch, home indicator ni nav bar
☐ Keyboard: Inputs visibles con teclado abierto
☐ Font scaling: Layout no se rompe al 200% font size
☐ Accessibility labels: Todos los botones e inputs tienen label
☐ Reduce motion: Animaciones se simplifican cuando está activo
☐ Gestos: iOS swipe back funciona. Android back button funciona
☐ Feedback: iOS = opacity fade. Android = ripple
☐ Haptics: Acciones clave tienen feedback háptico
☐ Offline: Vista de personal funciona sin conexión
☐ Empty states: Todas las listas tienen estado vacío con CTA
☐ Voz SEN: Textos en tuteo, español mexicano, sin jerga corporativa
