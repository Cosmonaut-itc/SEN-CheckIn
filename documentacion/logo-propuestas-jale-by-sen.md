# Propuestas de logo para `jale. by SEN`

Estas propuestas están pensadas para el slot compacto del header de marketing y dashboard.

## Archivos

- `apps/web/public/branding/logo-jale-proposal-01-j-check.svg`
- `apps/web/public/branding/logo-jale-proposal-02-jl-monogram.svg`
- `apps/web/public/branding/logo-jale-proposal-03-biometric.svg`

## Lectura de cada opción

### 01. `J + check`

- Prioriza claridad a tamaño chico.
- Une `jale` con validación/asistencia confirmada.
- Es la opción más fuerte si se quiere comunicar utilidad del producto desde el isotipo.

### 02. `JL` refinado

- Conserva continuidad con el uso actual de `JL`.
- Se siente más marca editorial que producto operativo.
- Funciona si se quiere mantener el rebrand con mínima fricción visual.

### 03. Biométrico abstracto

- Empuja más el lado tecnológico y de verificación facial.
- Evita una cara literal o una huella genérica.
- Es la opción más distintiva si se quiere construir un símbolo propio.

## Recomendación

Si el objetivo es mejorar rápidamente el branding sin perder legibilidad en móvil, la mejor ruta es `01. J + check`.

Si el objetivo es evolucionar hacia una identidad más propietaria y menos dependiente de iniciales, la mejor ruta a madurar es `03. Biométrico abstracto`.
